    Echarts 介绍#

    Echarts是基于 JavaScript 的开源可视化图表库, 详细图表配置可见官网。

    常用图:

    • 柱状图
    • 折线图
    • 饼图
    • 散点图
    • 热图

  methods: {
    draw_pie_chart() {
      var toolboxShow = true

      var legendposition_y = 'center'
      var title = [
        {
          text: 'Sub-population pie charts',

              param.seriesName +

              param.data.name.replace(/\s\(.*?\)/g, '') +
              ' : ' +
              String(param.data.value) +
              ' (' +
              String(param.percent) +

              show: true,
